\subsection*{Introduction}

We generate 3D gaussian data,
We generate 3D Gaussian data,
\begin{Schunk}
\begin{Sinput}
R> set.seed(1)
R> n <- 100
R> x <- rnorm(n); y <- 2*x + rnorm(n)/2
R> U3 <- cbind(x, y, z = -3*x + y + rnorm(n)/4)
\end{Sinput}
\end{Schunk}

\begin{Schunk}

R> str(U3) # its structure ((comment kept))
